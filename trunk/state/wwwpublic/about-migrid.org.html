About MiGrid

Minimum intrusion Grid (MiG or MiGrid) is a data storage, sharing and processing facility provided by University of Copenhagen to employees, students and partners. MiGrid delivers centralized storage space for personal and shared image files in addition to processing data.

Sensitive and Personal Data (GDPR)

Please note that while MiGrid comes with a strong security focus, it is only a solution approved for general scientific data and not for highly sensitive data. In particular it is not for personal data classified as sensitive in the EU Regulation 2016/679 (General Data Protection Regulation). In case you need to work on such sensitive data we recommend the MiGrid sister facility SIF, which is intended and approved for exactly that purpose.

Om MiGrid

Minimum intrusion Grid (MiG eller MiGrid) er et lager-, delings- og processerings-system leveret af Københavns Universitet til ansatte, studerende og partnere. MiGrid leverer centraliseret lagerplads for personlige - og delte data samt faciliteter til processering af pågældende data.

Følsomme - og personhenførbare data (GDPR)

Bemærk venligst at selvom MiGrid har fokus på stærk sikkerhed, så er det kun en løsning til generelle forskningsdata og ikke til følsomme data. Herunder specielt ikke personhenførbare data klassificeret som følsomme i EUs persondataforordning (forordning 2016/679, også kendt som GDPR). Såfremt du har behov for at arbejde på sådanne sensitive data anbefaler vi i stedet at benytte MiGrids søster-facilitet SIF, som netop er beregnet og godkendt til det formål.
