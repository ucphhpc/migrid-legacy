MODI provides a cluster of 8 compute nodes that can be utilized for long running batch jobs.
The system is configured as a SLURM cluster in which regular slurm commandscan be utilized.
This includes functionality such as 'sbatch', 'squeue', 'sinfo' and friends.

The current setup does not at the moment support interactive functionalities such as 'srun' and similar.
