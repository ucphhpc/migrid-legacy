MiG Virtual Machine Images

OS Images

Download an OS image pack tarball to provide VM support on a resource or pick individual images in the sub folders for private use. The data images are used to generate user VMs on MiG servers.

Steps to Run a VM

  • Install virtualbox (apt-get install virtualbox-ose virtualbox-ose-dkms)
  • Download one or more OS and data images to ~/.VirtualBox/HardDisks/
  • Download the vbox helper script
  • Launch it with the image details like
    bash vbox31-display.job i386 escience-base 12.04
    for the ubuntu-12.04-i386-escience-base.vmdk and ubuntu-12.04-i386-data.vmdk images with VirtualBox 3.x or 
    bash vbox40-display.job amd64 escience-astro 12.04
    for the ubuntu-12.04-amd64-escience-astro.vmdk and ubuntu-12.04-amd64-data.vmdk images with VirtualBox 4.x.
The instructions were tested with VirtualBox 3 on Ubuntu and with VirtualBox 4 on Mint Debian, and may work for other distributions.
